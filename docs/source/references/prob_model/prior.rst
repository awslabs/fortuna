Prior distribution
==================
We support Gaussian prior distributions,
specifically :class:`~fortuna.prob_model.prior.gaussian.IsotropicGaussianPrior`
and :class:`~fortuna.prob_model.prior.gaussian.DiagonalGaussianPrior`.

Alternatively, you can bring your own prior distribution by overwriting the abstract class
:class:`~fortuna.prob_model.prior.base.Prior`. Please find the references below.
